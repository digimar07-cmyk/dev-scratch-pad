# Laserflix

## Referência
O arquivo `laserflix_v740_Ofline_Stable.py` é o código original completo e serve como referência mestre para consultas e comparações futuras. Não deve ser alterado.

## Estrutura
- `main.py` — Ponto de entrada
- `config.py` — Constantes globais
- `app.py` — Inicialização e conexão dos módulos
- `core/` — Dados e estado
- `services/` — Lógica de back-end (sem Tkinter)
- `ui/` — Interface (Tkinter)
